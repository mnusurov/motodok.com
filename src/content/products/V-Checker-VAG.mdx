Автомобильный сканер V-Checker VAG&nbsp;для всех автомобилей (без CAN шины) производства концерна VAG (VW, Audi, Skoda) , позволяет читать\стирать коды ошибок, отображает коды описание кодов неисправностей (DTC), выводит инженерные параметры в реальном времени. Невысокая цена V-Checker VAG делает его доступным автолюбителям.

Цена: 80$

Купить V-Checker VAG

Если у вас загорелся на панели индикатор неисправности "Check Еngine"&nbsp;&nbsp;вам нужен этот прибор! V-Checker VAG небольшой прибор умещающийся в отделении для перчаток (бардачке) предназначенный для чтения и стирания кодов неисправностей (Diagnostic Trouble Codes или сокращенно DTC). Прибор способен выводить инженерные параметры в реальном времени (Live Data или Data Stream). Малый размер, надежная конструкция, привлекательная цена простота в использовании - идеальное сочетание для автолюбителей. V-Checker VAG автономный прибор, не требующий компьютера для работы.

Просто подключите в OBD2 разьем V-Checker VAG, включите зажигание (можно завести двигатель - для параметров реального времени) и прибор готов к чтению\стиранию ошибок. Описание кода неисправности отображается на экране, нет нужды искать его по коду ошибки в документации. Коды выводятся по кругу на экран и пролистываюстся кнопками [PREVIOUS] или [NEXT]. Это важное преимущество прибора по сравнению со сканерами отображающие только код ошибки.

Диагностические возможности

Большой LCD экран: описание кода неисправности выводится непосредственно на экран, нет необходимости искать его в документации

Коды выводятся по кругу нажатием кнопок [↑] или [↓] - большое преимущество по сравнению со старыми сканерами кодов ошибок (старые сканеры пытаются стереть код ошибки сразу его после отображения)
