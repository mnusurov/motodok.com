Диагностический сканер **BMW ICOM** (Integraget Communictaion Optical Module) является новейшей платформой диагностики третьего поколения (3G), пришедшей на смену  устаревшим дилерским приборам [BMW GT1](/diagnosis/bmw-gt1/), [BMW OPS](/diagnosis/bmw-ops/) и OPPS. **BMW ICOM** работая совместно с программным обеспечением **BMW ISTA-D** (Integrated Service Technical Application for Diagnosis) и  **BMW ISTA-P** (Integrated Service Technical Application for Programming) установленным на **BMW ISID** (Integrated Service Information Display) обеспечивает более высокую производительность по сравнению **BMW DIS**, **BMW SSS**. Наиболее важными преимуществами платформы  третьего поколения является: полная поддержка всех поколений автомобилей BMW включая F серию; совместимость с шинами FlexRay, K-CAN, PT-CAN, ZGM; поддержка оптического интерфейса MOST-Bus (Fiber Optic System).  Читает и сбрасывает (стирает) ошибки, считывает инженерные параметры, выполняет тест исполнительных устройств, индентифицирует версии блоков и их программного обеспечения, кодирует и программирует блоки управления.

## Описание компонентов платформы третьего поколения диагностики BMW

**BMW ISID** (Integrated Service Information Display) — диагностическая станция

Информационный дисплей интегрированного обслуживания полностью заменяет сканеры GT1 и OPS/OPPS. Все требуемые диагностические операции могут проводиться с помощью этого прибора, также поддерживаются функции кодирования и программирования. При наличии необязательного центрального сервера ISIS, ISID может получать всю необходимую информацию от центрального сервера через сеть. В нашем предложении его роль выполняет планшетный ноутбук Lenovo ThinkPad X61 c тачскрином с установленным программным обеспечением BMW ISTA-D и ISTA-P. Текущие версии программного обеспечения **BMW ISTA-D**, **BMW ISTA-P** а также преимущества нашей сборки можно посмотреть на странице [Обновление ПО для BMW ICOM](/services/bmw-gt1-update/)

**BMW ISTA-D** (Integrated Service Technical Application Diagnostics )

Cистема интегрированного обслуживания/диагностики является прикладным программным обеспечением комплекса ISID. ISTA/D включает в себя все необходимы функции для максимального быстрого поиска неисправностей. Используя ISTA/D для диагностики, Вы получите возможность использовать в своей работе следующие функции:

-   Короткий тест по дереву блоков управления
-   Диагностика через Функции ЭБУ (чтение и стирание ошибок, параметры в реальном времени, активации, сброс адаптаций)
-   Ведомый поиск неисправностей
-   Поиск неисправностей с помощью плана проверки и списка случаев
-   Доступ к технической информации по обслуживанию автомобилей, электросхемам и т.д.

**BMW ISTA-P** (Integrated Service Technical Application Programming )

Cистема интегрированного обслуживания и программирования является прикладным программным обеспечением комплекса ISPI и пришла на смену прежней системе Progman. ISTA/P содержит данные для программирования и кодирования автомобилей. Кроме того, в ней предусмотрены новые функции, которые понадобятся в будущем. С внедрением ISTA/P появились следующие новшества:

-   Конфигурируемый план мероприятий с графиками и таблицами
-   Подробный показ состояния блока управления
-   Запись кода допуска/FA с любого места СТОА, если есть доступ к ISTA/P
-   Дополнительное расширение и адаптация плана мероприятий
-   Создание списков для заказа блоков управления, если потребовалась замена АО
-   Прерывание сеанса при отсутствии запчастей
-   Отображение версии Progman или ISTA/P, с помощью которой автомобиль был запрограммирован в последний раз

С целью сделать наше предложение доступным широкому кругу диагностов и удешевления цены нашего предложения из него исключены следующие необязательные компоненты платформы третьего поколения диагностики BMW:

**BMW ISIS** (Integrated Service Information System)

Cервер интегрированного обслуживания, является основным сервером (ядром) программного комплекса последнего поколения, предназначенного для обслуживания всех подсистем. ISIS управляет работой и взаимодействием всех подсистем — ISSS, ISID, ICOM, ISAP, IMIB. Кроме того, сервер поддерживает электронный каталог запасных частей ETK, а также новые системы для процесса приемки и обслуживания автомобилей. В его отсутствие недоступно автоматическое обновление ПО для BMW ISID и ICOM.

**BMW ISSS** (Integrated Software Service Station)

Сервер применяется для кодирования и программирования электронных блоков управления BMW, их инициализации после процедуры программирования, управления CKM (car key memory — память ключа и машины), для выполнения операций дооснащения и переоснащения, пришедший на смену устаревшему Progman. В качестве интерфейса для подключения к автомобилю ISSS может использовать ТОЛЬКО диагностический модуль нового поколения ICOM. Старые интерфейсы (OPS, OPPS, Edic) не поддерживаются в ISSS. В оригинальном варианте ISID использует сервисы предоставляемые BMW ISSS для кодирования, программирования фактически являясь удаленным интерфейсом к серверу ISSS. В нашем варианте не требуется т.к. все необходимые сервисы ISSS работают локально на BMW ISID.

**BMW IMIB** (Integrated Measurement Interface Box)

Интеллектуальный измерительный интерфейсный блок представляет собой мощный измерительный прибор. При его отсутствии недоступна измерительная подсистема ISID и тест-блоки использующие ее.

**BMW ISAP** (Integrated Service Access Point)

Точка доступа интегрированного обслуживания представляет собой эффективное устройство для соединения любых систем диагностики и программирования с ISIS по беспроводной локальной сети. Для радиосвязи используется стандарт 802.11n Draft 2.0. ISAP работает в диапазоне 2,4 ГГц или 5 ГГц. В нашем предложении не требуется так как между ICOM и ISID используется проводное соединение локальной сети.
