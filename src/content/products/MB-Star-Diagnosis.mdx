Семейство приборов автодиагностики под общим названием **StarDiagnose** пришло на смену морально устаревшему прибору диагностики **ННТ**. Первым прибором из этого семейства был известный большинству **МТ**. Прибор предназначен для полной диагностики электронных систем на автомобилях **Mercedes**.

На настоящий момент существует более новая платформа диагностики автомобилей Mercedes Benz включая модельный ряд 2011 г. и выше, см.**[Mercedes Benz StarDiagnosis SDConnect](/Diagnosis/star-diagnosis-sdconnect/)**

Комплект прибора состоит из мультиплексора с диагностическими кабелями и ноутбука с установленным программным обеспечением состоящим из трех частей: **DAS** - основная часть системы, диагностическая программа, для старых машин автоматически запускаяется **HHT WIN** - эмулятор одноименного устаревшего диагностического прибора.  **WIS/ASRA**  - документация по ремонту и обслуживанию, электрические схемы, расчет норм времени. **SDMedia** – новый программный продукт, появился в середине 2006 года. Представляет собой мультимедийное руководство по поиску и устранению неисправностей. Кроме перечисленного программного обеспечения на **StarDiagnosis** дополнительно установлен каталог запасных частей **MB EPCNet** и каталог электрических схем с местоположением разьемов **StarFinder V3** .

Программное обеспечение может быть запущено на любом компьютере (отвечающий требованиям описанным в конце страницы), при покупке у нас доступно по вашему выбору в трех вариантах :

-   2,5'' HDD диск с установленным ПО для ноутбука **IBM ThinkPad T30**;
-   Образы виртуальной машины **VMWare** на 2,5'' HDD в корпусе c USB интерфейсом для работы на любом компьютере;
-   Образ жесткого диска **Acronis** на 2,5'' HDD в корпусе c USB интерфейсом для установки на любом компьютере при помощи входящего в комплект загрузочного CDROM **Acronis True Image with Universal Restore**;

Два последних варианта содержат инструкцию по установке и настройке системы для работы с образами системы с предустановленными **DAS**, **WIS\\ASRA**, **EPCNet**, **SDMedia**, **StarFinder** .

Текущие версии программного обеспечения можно посмотреть на нашей странице [Обновление для MB StarDiagnosis](/Services/mb-star-diagnosis-das-upd/).

Для тех кто предпочитает полный комплект готовый к работе - в качестве опции мы можем предложить б\\у ноутбук **IBM ThinkPad T30** (цена в конце страницы в разделе опции).
