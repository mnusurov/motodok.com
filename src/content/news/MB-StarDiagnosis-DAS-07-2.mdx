Текущая версия программного обеспечения **DAS 07/2009**

суббота, 15 августа 2009

Программное обеспечение при покупке у нас **[MB StarDiagnosis](/Diagnosis/mb-star-diagnosis/)** доступно по вашему выбору в трех вариантах :

-   2,5'' HDD диск с установленным ПО для ноутбука IBM T30
-   Образы виртуальной машины VMWare на 2,5'' HDD в корпусе c USB интерфейсом для работы на любом компьютере (обязательно наличие COM порта)
-   Образ жесткого диска Acronis на 2,5'' HDD в корпусе c USB интерфейсом для установки на любом компьютере при помощи входящего в комплект загрузочного CDROM Acronis True Image with Universal Restore (обязательно наличие COM порта)

Все три варианта содержат следующие версии программное обеспечения:

-   DAS 07/2009
-   WIS/ASRA 07/2009
-   EPCNet 07/2009
-   SDMedia
-   StarFinder V3

В комлекте имеется лицензионный ключ с инженерным меню (Developer Mode)

Если у вас уже есть прибор [**MB StarDiagnosis**](/Diagnosis/mb-star-diagnosis/), мы можем предложить вам обновление программного обеспечения StarDiagnosis DAS до 07/2009.
