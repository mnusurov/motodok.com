VAG-COM: Перекодирование (Recode)


VAG-COM
Кодирование
(VAG 1551/1552 функция 07)

Кодирование используется для установки различных опций
в Модуле Управления.

Предупреждение!
Вы должны обратиться к Рабочему руководству для вашего автомобиля перед попыткой
перекодировать Модуль Управления.  По крайней мере запишите
первоначальные значения. Нет никакого способа "отменить"
или восстановить первоначальные значения, если Ваша попытка не удалась.

Замечание:
Некоторые Модули требуют правильного Логина перед
кодированием их.

Уникальная особенность VAG-COMа:
Вы можете оставить Work Shop Code неизменным,
или ввести любой рабочий код которого Вы желаете.
Некоторые другие программы всегда покидают сигнальный
WSC, когда Вы кодируете модуль.

Используйтекнопки Do It! or Cancel, чтобы возвратиться
к Controller Info / Select Function.
