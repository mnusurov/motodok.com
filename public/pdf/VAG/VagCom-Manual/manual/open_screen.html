VAG-COM
Открытие Контроллера / Выбор Функций

Этот экран появится, когда VAG-COM
попытается установить связь с
модулем Управления.

Comm Status
Показывает состояние связи текущей сессии как только связь установлена,
IC= показывает количество раз, которое сессия была инициализирована. Если IC=будет равно более 1, это означает, что связь достоверна менее чем на 100 %.
Blk= является счетчиком, который показывает текущий
номер блока протокола. Вы можете игнорировать это;
он должен постоянно увеличиваться и пройти
через 255.  Если он остановился, то это означает, что была обнаружена ошибка связи.
Подождите несколько секунд,
сессия должна автоматически повторно инициализироваться.
TE= и RE= являются счетчиками для ошибок в пределах
индивидуальных пакетов.

Информация Контроллера
Если однажды сессия связи была установлена,
все данные "Идентификации" Модулей Управления представлены здесь.
VAG Number является номером части VW/Audi для этого контроллера.
Component обычно говорит то, чем это является и может содержать
номер версии для внутренних встроенных программ контроллера.
Sort. Coding является версией Программного обеспечения и определяет
различные варианты для контроллера.
Shop # опознает WorkShopCode (рабочие коды), запомненные
scan-tool при Перекодировке Модуля Управления.
Замечание: Некоторые старые Модули Управления — не кодируются
и Вы можете увидеть part number Bosch или другую
информацию в данных областях.
Extra (дополнительные) области используются, чтобы выделить номера частей
для любых "ведомых" контроллеров. Пример: Центральные Контроллеры Комфорта на более новых VW имеют до четырех
"ведомых" контроллеров (один в каждой двери).

Базовые Функции
Эта группа функций "Safe" используется для чтения различных
данных от Модули Управления. Щелкните на каждой из кнопок
за дополнительной информацией относительно индивидуальных функций.
Недоступные кнопки - функции, которые еще не были
осуществленны.

Расширенные Функции
Эти функции могут программировать изменения
в Модулях Управления.
Вы должны обратиться к Рабочему руководству
вашего автомобиля перед "игрой" с этими функциями.
Вы можете щелкнуть на каждой из кнопок в этом
демонстрационном примере для получения дополнительной информации об
индивидуальных функциях.

Используйте Close Controller, Go Back для должным образом
закрытия сессии связи и возвращения к
Выбору Модуля Управления
